# Prescription Scheduler

This repository contains a full-stack solution for processing prescription schedules.

## Stack
- Client: React + TypeScript
- Server: Express + TypeScript

## Stages (from test document)
- **Stage 1**: Collect user inputs (availability, prescription type, dosage details)
- **Stage 2**: Process answers to generate a 14-day dosage schedule, including handling:
  - Unavailable days
  - UK bank holidays

## Repository Structure
- `/client` - React (TypeScript)
- `/server` - Express (TypeScript)
